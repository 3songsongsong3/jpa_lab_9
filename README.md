# JPA 스프링 웹 어플리케이션 jpa_lab_9

## 진행 순서
1. 프로젝트 환경 설정
2. 도메인 모델과 테이블 설계
3. 어플리케이션 기능 구현

## 사용 기술
* 뷰 : JSP, JSTL
* 웹 계층 : 스프링 MVC
* 데이터 저장 계층 : JPA, 하이버네이트
* 프레임워크 : 스프링 프레임워크
* 빌드 : 메이븐
